🎯 PRIMARY GOAL

🎨 DESIGN SYSTEM (IMPORTANT)

* Primary accent: neon green (#2DFF88) for profit/wins

* Neutral text: white / light gray
* Cards: dark elevated panels with soft borders

Style Rules:

* Minimal clutter
* Strong spacing
* Large readable stats
* Rounded cards (12–16px radius)
* Subtle shadows
* Smooth hover animations
* Mobile-first layout
